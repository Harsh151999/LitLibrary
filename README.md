#LitLibrary

A simple Python-based library management system for managing a collection of books. This project demonstrates object-oriented programming with classes for books and a library, including features like adding, searching, filtering, and displaying books.

Features
Add Books: Easily add new books with details like name, author, genre, rating, and price.
Search Books: Search by title, author, or genre.
Display Books: View all books or filtered lists.
Highest Rated: Find the top-rated book(s).
Price Filtering: Filter books within a price range.
Statistics: Get overall library stats, including total value, average rating, and genre distribution.
